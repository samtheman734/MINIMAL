# MINIMAL:

**MINIMAL** is a straightforward programming language interpreter written in Python. It supports basic features such as variable assignment, simple arithmetic operations, and printing output. Designed for experimentation and personal use, MINIMAL provides an easy way to create and run simple scripts.

---

## Features

- Variable assignment (e.g., `x = 5`)
- Basic arithmetic (`+`)
- Printing output (`print x`)
- Interactive mode (REPL) for live coding

---

## Getting Started

### Prerequisites

- Python 3.x installed on your system

### Running MINIMAL

1. Download or clone this repository.
2. Run the interpreter:
```bash
python minimal.py
